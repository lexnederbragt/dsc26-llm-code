# LLM-powered Programming and Data Analysis

A Digital Scholarship Days 2026 [workshop](https://www.ub.uio.no/english/courses-events/events/dsc/dsdays/2026/12-llm-powered-programming-and-data-analysis-20260106)

## To generate the  HTML version

* Requires quarto and pandoc installed
* Run `quarto render` to get the html version locally
* Run `quarto publish gh-pages` to update the [site on GitHub Pages](https://lexnederbragt.github.io/dsc26-llm-code)
